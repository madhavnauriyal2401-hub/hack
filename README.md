# 🛡️ Raksha Sutra (रक्षा सूत्र)
### *A Digital Guardian for Indian Elders*

**Raksha Sutra** is a specialized security application designed to protect senior citizens from the rising tide of digital fraud, fake news, and phishing attacks in the Indian digital landscape. 

Developed by **Madhav Nauriyal** and **Aarav Sharma** (1st Year B.Tech, Graphic Era Deemed to be University), this project leverages the power of Gemini AI to provide a caring, elder-centric defense mechanism.

---

## ✨ Key Features

- **🧠 Gemini AI Analysis**: Real-time verification of news, videos, payment requests, and suspicious links.
- **👁️ Eye-Soft Design**: A custom-engineered UI using warm paper and sage tones to reduce glare and eye strain for senior users.
- **🇮🇳 Localized Fraud Detection**: Specifically trained to recognize Indian scam patterns (KBC lotteries, electricity bill threats, UPI phishing).
- **🚨 Victim Support Roadmap**: Direct integration with the **Government of India's 1930 Cyber Helpline** and official reporting portals.
- **📚 Safety Playbook**: Easy-to-understand guides on UPI safety, card security, and digital literacy.

## 🛠️ Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **AI Engine**: Google Gemini API (@google/genai)
- **Icons**: Lucide React
- **Visualization**: Recharts
